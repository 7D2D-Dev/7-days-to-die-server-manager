## Env vars

To start the application, you need to add these environment variables;

 - CSMM_HOSTNAME
 - API_KEY_STEAM
 - DISCORDBOTTOKEN
 - PAPERTRAILHOST
 - PAPERTRAILPORT

To test the application, add these env vars;

 - CSMM_TEST_IP
 - CSMM_TEST_WEBPORT
 - CSMM_TEST_TELNETPORT
 - CSMM_TEST_TELNETPW
 - CSMM_TEST_STEAMID
 - CSMM_TEST_STEAM_USERNAME
 - CSMM_TEST_STEAM_PASSWORD
 - CSMM_TEST_AUTHNAME
 - CSMM_TEST_AUTHTOKEN
 - DISCORDTESTCHANNEL
 - DISCORDTESTGUILD
